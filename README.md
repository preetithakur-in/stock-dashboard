# 📊 Stock Dashboard

A Python-based stock market dashboard built in Jupyter Notebook to fetch, analyze, and visualize stock data. This project uses real market data to explore trends, returns, and moving averages.

## 🚀 Features

* 📥 Fetches real-time/historical stock data using `yfinance`
* 📈 Visualizes stock price trends using Matplotlib & Seaborn
* 📊 Calculates daily returns for performance analysis
* 📉 Implements moving averages for trend insights
* 🧮 Clean and structured data analysis using Pandas

## 🛠️ Tech Stack

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Seaborn
* yfinance
